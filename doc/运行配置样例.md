TLS broker 示例：

```json
"broker": "ssl://broker.example.com:8883",
"tls": {
  "enabled": true,
  "caFile": "/opt/modbus-gateway/config/runtime/tls/ca.crt",
  "certFile": "",
  "keyFile": "",
  "insecureSkipVerify": false
}
```

如果 broker 要求客户端证书，`certFile` 和 `keyFile` 必须同时填写；生产环境禁止开启 `insecureSkipVerify`。
